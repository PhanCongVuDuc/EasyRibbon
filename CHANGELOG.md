# Changelog

All notable changes to EasyRibbon will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial public release
- Attribute-based ribbon UI definition system
- Support for Button, Panel, Tab attributes
- Support for StackedButton and PulldownButton
- Module system for organizing multiple add-ins
- ModuleRegistry for managing modules
- Support for Revit 2021, 2023, and 2024
- Extension methods for easier API usage
- Complete example projects
- Documentation and README
